\section{Project Goal}

The project studies speech explainability for emotion recognition on IEMOCAP. The planned pipeline compares:

\begin{enumerate}
  \item SpeechXAI segment-level explanations.
  \item A LeGrad-inspired temporal attribution method based on gradients with respect to Transformer attention maps.
\end{enumerate}

Both methods will be converted to ranked time intervals in seconds and evaluated using the same deletion protocol. For each audio sample, selected regions are replaced by silence while preserving the original audio length. The main metric is the confidence drop for the original predicted class:

\[
\mathrm{drop} = p_{\mathrm{original}}(\hat{y}) - p_{\mathrm{masked}}(\hat{y}).
\]

\section{Corrected Quantitative Evaluation}

The quantitative deletion protocol was updated after the initial scaffold. The old fixed-percentage masking budgets are no longer the project protocol. Because SpeechXAI naturally produces word-level explanations, SpeechXAI now defines the masking duration.

For each audio sample and each $k \in \{1,2,3,5\}$, the top-$k$ SpeechXAI words are selected and their total duration $X$ is computed. SpeechXAI masks exactly those word intervals. LeGrad masks its highest-scoring time bins whose total duration equals $X$. The random baseline masks random time bins with the same total duration $X$, repeated over multiple random trials. This keeps the classifier, audio sample, masking operation, and amount of removed audio fixed across methods; only the selected regions differ.

The deletion-results table should therefore include:

\begin{verbatim}
audio_id, k, method, masked_duration, original_label,
original_confidence, masked_confidence, drop, relative_drop,
prediction_flipped, random_trial
\end{verbatim}
